In this project, we are creating a 3D-model of a Classroom. 

To run the 3D-classroom, use the following commands in 
the Terminal:

$ cmake .
$ make all				//create an output file
$ ./Classroom 		//run the output file

$make clean 			//removes the all object file that was created
